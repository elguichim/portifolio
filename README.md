# Portifolio

Este é o repositório do meu portfólio pessoal.  
Aqui estão os códigos e arquivos que compõem o site.

## 🚀 Publicação

### GitHub Pages
1. Vá em **Settings → Pages** no repositório.  
2. Escolha a branch `main` e a pasta `/root`.  
3. Salve.  
👉 O site ficará disponível em:  
https://elguichim.github.io/portifolio/

### Vercel
1. Acesse [Vercel](https://vercel.com) e faça login com GitHub.  
2. Clique em **New Project** e selecione este repositório.  
3. Configure:
   - **Root Directory** → `front/` (se o projeto está lá).  
   - **Build Command** → `npm run build`  
   - **Output Directory** → `dist`  
4. Clique em **Deploy**.  
👉 O site ficará disponível em:  
https://portifolio.vercel.app
